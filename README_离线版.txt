中日贸易包税报价工具（离线单机版）
================================

使用方法：
  1. 解压整个压缩包到一个文件夹
  2. 双击 standalone_offline.html 用浏览器打开
  3. 完全离线，不需要联网，HS 编码数据库已内嵌

说明：
  - 本版本为纯离线版，HS 税率数据已打包进文件（约 6532 条）
  - 请用 Chrome / Edge / 360 等现代浏览器打开
  - 如打开后中文显示乱码，请确认浏览器编码设为「自动检测」

更新记录：
  - 已修复货值币种（CNY/USD）未统一换算为日元即参与计算的 bug
  - 文件已统一为 UTF-8 编码，避免中文乱码
